REFERENCES

Hyphe is a free open source software developed by the médialab team at Sciences Po and released on GitHub under AGPL 3.0 license. Its development was initially financed thanks to the DIME-Web project, part of DIME-SHS research equipment financed by the EQUIPEX program (ANR-10-EQPX-19-01).

Hyphe's objectives and methodology are described in two academic papers that can be used as reference whenever publishing scientific results which involved building web corpora using Hyphe:
  • JACOMY, Mathieu, GIRARD, Paul, OOGHE-TABANOU, Benjamin, et al, "Hyphe, a curation-oriented approach to web crawling for the social sciences.", in International AAAI Conference on Web and Social Media. Association for the Advancement of Artificial Intelligence, 2016. https://spire.sciencespo.fr/hdl:/2441/6obemb2hsj9pboj9bbvc7sftne/resources/jacomy-all-hyphe-icwsm-2016.pdf
  • OOGHE-TABANOU, Benjamin, JACOMY, Mathieu, GIRARD, Paul & PLIQUE, Guillaume, "Hyperlink is not dead!" (Proceeding / Slides), In Proceedings of the 2nd International Conference on Web Studies (WS.2 2018), Everardo Reyes, Mark Bernstein, Giancarlo Ruffo, and Imad Saleh (Eds.). ACM, New York, NY, USA, 12-18. DOI: https://doi.org/10.1145/3240431.3240434

HYPHE BROWSER

Hyphe Browser or "HyBro" is a separate application that connects to the same server. It complements the web interface and allows you to build your corpus while browsing the web.
